## 核心问题

如何通过连续追问，把浅层提问升级为高价值问题。

## 这节学什么

学习把“我要答案”转成“我要看清问题结构、前提和替代方案”。

## 为什么这节重要

很多人在用大模型时，提问方式仍然停留在“帮我总结”“帮我分析”“给我一个答案”。这样做并不总是错，但对于复杂任务来说，这往往只能得到表面上通顺、实际上深度不足的回答。

真正高阶的人机协作，不是把 AI 当搜索框，而是把它当推理伙伴。你不是只要答案，而是要它帮助你：

- 定义问题边界
- 暴露隐含假设
- 比较不同解释
- 设计下一步行动

这就是战略提问。

## 关键概念

- question ladder
- scope
- assumptions
- counterexample

## 通俗解释

### 1. 什么是战略提问

战略提问不是“问得更复杂”，而是“问得更有方向”。它不急着让模型给结论，而是要求模型先帮你看清：

- 这个问题的边界是什么
- 这个问题的关键变量是什么
- 可能还有哪些替代解释
- 结论成立依赖哪些前提

### 2. 为什么高质量问题比答案更重要

如果问题本身问错了，得到再漂亮的答案也没有意义。

例如：

```text
帮我分析一下为什么客户流失了。
```

这个问题太宽，会让模型输出一堆泛泛原因。但如果你问：

```text
请基于最近三个月华东区数据，分析客户流失的可能原因，并列出至少三种替代解释，以及每种解释需要验证的证据。
```

模型就更容易进入真正有用的分析状态。

### 3. 什么是问题分层

问题可以粗略分成三层：

#### 第一层：功能性提问

主要目的是获取信息。

例如：

- 什么是 RAG
- 请总结这篇文章

#### 第二层：分析性提问

主要目的是建立结构、比较差异、寻找关系。

例如：

- 这三个理论框架有什么差异
- 这两种方法适合的场景分别是什么

#### 第三层：战略性提问

主要目的是驱动推理、暴露假设、发现盲点、支持决策。

例如：

- 如果这个结论不成立，最可能的替代解释是什么
- 当前方案最大的风险盲区是什么
- 这个研究问题真正的分歧点在哪里

### 4. 什么是边界限定

边界限定就是告诉模型：这个问题只在什么范围内成立。

边界可以包括：

- 时间范围
- 空间范围
- 对象范围
- 学科视角
- 任务目标

边界越清楚，模型越不容易空泛。

### 5. 什么是替代解释

替代解释就是要求模型不要只给出一个“看起来合理”的结论，而是主动提出其他可能性。

例如：

- 客户流失可能是价格问题
- 也可能是竞争对手促销
- 也可能是产品体验下降

这一点非常重要，因为大模型很容易过早锁定某一个解释。

### 6. 什么是追问链

高质量问题通常不是一步到位，而是通过追问逐步升级。

一个常见追问链可以是：

1. 先澄清问题
2. 再拆分结构
3. 再挑战结论
4. 最后导向行动

### 7. 什么是问题升级

问题升级就是把原本浅层的问题提升到更值得思考的层次。

例如：

- 从“怎么做好一门课”
- 升级到“这门课真正要训练的能力是什么”
- 再升级到“什么样的课程结构最能支持这种能力形成”

## 典型练习

针对一个课程主题设计 10 个逐层递进的问题。

## 最小示例

先问“怎么学提示词工程”，再追问“目标场景是什么、成功标准是什么、常见失败是什么”。

## 8个可直接上手的例子

### 例子1：课程设计

低质量提问：

```text
帮我设计一个时间管理培训课。
```

战略提问：

```text
请为职场新人设计一门2小时的时间管理培训课。
边界限定：纯线上授课，不依赖线下道具。
要求：列出3个核心模块、互动方式和可能导致参与度下降的原因，并提供替代活动方案。
```

### 例子2：研究选题

低质量提问：

```text
最近在线教育有什么好题目？
```

战略提问：

```text
我的研究兴趣是“混合式学习中的成年在职学习者”。
请提出3个可研究问题。
要求：每个问题都说明核心变量、替代解释、可能的数据验证方式。
```

### 例子3：文献分析

低质量提问：

```text
帮我总结这三篇文献。
```

战略提问：

```text
请比较这三篇文献在理论框架和方法路径上的差异。
要求：指出真正的分歧点、共同忽略的边界条件，以及由此引出的下一步研究问题。
```

### 例子4：方案评估

低质量提问：

```text
看看我的项目方案怎么样。
```

战略提问：

```text
请扮演恶魔辩护人，对以下项目提案进行漏洞审查。
要求：指出最致命的两个风险盲区、证据不足的主张，并给出最小修复建议。
```

### 例子5：项目规划

低质量提问：

```text
帮我写一个新产品上线计划。
```

战略提问：

```text
我们计划下个月上线一款新产品。
请先拆出3个关键里程碑，再指出执行这些里程碑时可能遇到的阻力，并给出应对方案。
```

### 例子6：教学设计

低质量提问：

```text
教我怎么讲牛顿第二定律。
```

战略提问：

```text
请为高中课堂设计一条3到4步的追问链，引导学生自己理解牛顿第二定律。
边界限定：不用微积分，贴近日常生活。
要求：同时列出学生可能的错误理解与应对话术。
```

### 例子7：写作改进

低质量提问：

```text
帮我把这段话改专业一点。
```

战略提问：

```text
请先检查下面这段引言是否存在逻辑漏洞、过度概括或隐含假设，再给出修改版本。
要求：修改时明确收窄主张边界。
```

### 例子8：决策分析

低质量提问：

```text
公司应该出海还是继续做国内市场？
```

战略提问：

```text
请从竞争环境、成本收益和风险三个维度比较“出海”和“继续深耕国内市场”。
要求：列出你的核心假设，并说明如果这些假设不成立，决策应如何调整。
```

```python
2. 暴露假设
3. 提出替代解释
4. 引导下一步行动

原始问题：
{question}
"""

    payload = {
        "model": MODEL,
        "prompt": prompt,
```

### 方式六：保存战略问题实验结果

```python
import csv
import requests

URL = "http://localhost:11434/api/generate"
MODEL = "gemma3:4b"

questions = [
    "帮我设计一个课程。",
    "帮我总结一下这几篇文献。",
    "帮我分析一下为什么客户流失了。"
]


def upgrade_question(question):
    prompt = f"""
请把下面这个问题升级成战略提问。

要求：
1. 明确边界
2. 暴露假设
3. 提出替代解释
4. 引导下一步行动

原始问题：
{question}
"""

    payload = {
        "model": MODEL,
        "prompt": prompt,
        "stream": False,
        "options": {
            "temperature": 0.4,
            "top_p": 0.9
        }
    }
    response = requests.post(URL, json=payload, timeout=120)
    response.raise_for_status()
    return response.json()["response"].strip()


rows = []
for q in questions:
    rows.append({

        "raw_question": q,

    writer.writeheader()
```

## 课堂练习方案

### 练习目标

帮助学生从“直接索要答案”转向“通过提问设计推理路径”。

### 练习材料

给学生一条模糊问题：

```text
老板让我分析一下为什么最近客户流失率变高了，帮我写个分析报告。
```

### 练习步骤

1. 找出这个问题缺少哪些边界
2. 要求加入至少三种替代解释
3. 要求模型说明每种解释该如何验证
4. 对比重写前后的输出差异

### 观察点

- 问题是否更聚焦
- 模型是否开始真正分析，而不只是堆结论
- 是否出现了替代解释
- 是否给出了下一步验证动作

## 课堂流程

1. 展示平面问题和高质量问题的差异
2. 把问题按层级分类
3. 练习追问与改问

## 教师提醒

### 提醒1

问题不是越长越好，关键是逻辑要素是否齐备。

### 提醒2

AI 给出的分析不是最终真理，而是帮助你搭建思考框架。

### 提醒3

如果第一轮回答不够深入，不要急着放弃，追问链往往比重开新对话更有效。

### 提醒4

高质量协作不是让 AI 包揽全部推理，而是你先搭框架，再让 AI 扩展和挑战。

## 常见误区

- 误以为问题越长越高级
- 误以为 AI 第一次回答就是最终答案
- 误以为不满意就重开，而不是继续追问
- 误以为所有问题都只需要一个结论
- 误以为替代解释是多余的
